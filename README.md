# Python for Hackers - Justin Seitz & Tim Arnold books insipred repo

This repository contains Python scripts inspired by topics commonly studied in cybersecurity, ethical hacking, and penetration-testing.
The exercises start from the very basics of Python and gradually progress toward more advanced, security-oriented projects.

## Structure
- `Networking basics/` 
- `Web exploitation/` 
- `Windows Trojans/` 
- `File exfiltration/` 
- `Windows privilege escalation/`
- `Offensive forensics/`
